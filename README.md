# Student Gradebook Management System

## Project Description

This project is a simple Student Gradebook Management System developed using Python and Object-Oriented Programming (OOP). It allows users to manage students, courses, assessments, and grades through a menu-driven program.

The project was created as part of a Python Programming assignment.

---

## Features

- Add new students
- Add new courses
- Enroll students in courses
- Add assessments (Quiz, Exam, Project)
- Record student grades
- Search for students by ID or name
- Display a student's report
- Delete students
- View all students
- View all courses
- Display a dashboard showing system statistics

---

## Object-Oriented Programming Concepts Used

This project demonstrates the following OOP concepts:

- Classes and Objects
- Encapsulation
- Inheritance
- Method Overriding (Polymorphism)
- Constructors
- Dictionaries and Lists
- User-defined Methods
